HTML Table - Cell Padding Cell padding is the space between the cell edges and the cell content. By default the padding is set to 0. To add padding on table cells, use the CSS padding property: Example th, td { padding: 15px; } th, td { padding-top: 10px; padding-bottom: 20px; padding-left: 30px; padding-right: 40px; } table { border-spacing: 30px; }
